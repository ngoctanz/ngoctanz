# Setup

1. Create a public GitHub repository named `ngoctanz`.
2. Copy `README.md`, `assets/`, and `.github/` into the repository root.
3. Push to the `main` branch.
4. Open the **Actions** tab and run **Generate contribution snake** once.
5. Confirm that an `output` branch is created. The snake animation will then appear in the README.

The local animated SVG files embed their PNG backgrounds, so they render without extra hosting.
